# NeuroFlow

NeuroFlow is an advanced system involving ingestion, retrieval, generation, evaluation, and fine-tuning subsystems.

## Folder Structure

- `backend/` - Core API and business logic.
- `frontend/` - User interface.
- `pipelines/` - Data ingestion and processing pipelines.
- `evaluation/` - Automated evaluation subsystem.
- `infra/` - Infrastructure and deployment configuration.
- `docs/` - Architecture, API contracts, and decision records (ADRs).
